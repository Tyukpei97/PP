# Задание 1 — Редактор фигур (WinForms, .NET 8)

## Как запустить
1. Откройте `Task1_ShapeEditor.sln` (или прямо `ShapeEditor.WinForms.csproj`) в Visual Studio 2022.
2. Запустите проект **ShapeEditor.WinForms**.

## Возможности
- Рисование фигур: **Прямоугольник / Эллипс / Линия**
- Выделение, перетаскивание
- Изменение размера: маркеры по углам (для линии — концы)
- Цвет линии, цвет заливки (для прямоугольника/эллипса), заливка вкл/выкл
- Толщина линии
- Слои: «Вперёд» / «Назад»
- Сохранение и загрузка в `*.json`

## Горячие клавиши
- Ctrl+N — новый
- Ctrl+O — открыть
- Ctrl+S — сохранить
- Delete — удалить выделенное
- Esc — отменить рисование
- Ctrl+Up / Ctrl+Down — слой вверх / вниз
